# 字符串 格式化

* 字符串格式化符号含义
```python
>>>“{0} love {1}.{2}”.format("I" ,"ctoedu", "com")
'I love ctoedu.com'

>>>“{a} love {b}.{c}”.format(a="I" ,b="ctoedu", c="com")
'I love ctoedu.com'

>>>"{0:.1f}".format(27.658, 'GB')
'27.7GB'

>>> '%c' % 97
'a'

>>> '%c $c %c' % (97, 98, 99)
'a b c'

>>> '%s' % 'I Love CtoEdu.com'
'I Love CtoEdu.com'

>>> '%d + %d = %d' % (4, 5, 4+5)
'4 + 5 = 9'


>>> '%o' % 10
'12'

>>> '%x' % 10
'a'

>>> '%X' % 10
'A'

>>> '%f' % 27.658
'27.658000'

>>> '%5.1f' % 27.658 # 表示总长度为5 且小数点一位
' 27.7'

>>> '%10d' % 5 # 表示中长度为10
'         5'
```


## 字符串格式化符号含义及转义字符含义

> 字符串格式化符号含义：Python2.X 与Python3有区别，去掉了%p，%u

```xml

%c 格式化字符及ASCCII码
%s  格式化字符串
%d 格式化整数
%o 格式化无符号八进制
%x 格式化无符号十六进制数
%X 格式化无符号十六进制数（大写）
%f 格式化定点数，可指定小数点后的精度
%e 用科学计数法格式化定点数
%E 作用同%e，用科学技术法格式化定点数
%g 根据值的大小决定使用%f或%e
%G 作用同%g，根据值得大小决定使用%f或者%e

```
